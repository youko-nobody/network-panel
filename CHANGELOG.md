# Changelog

## 1.0.7 - 2026-06-14

- 修复重启后首页线路名称一直显示第一条的问题。
- 修复从设置页返回首页后线路视图引用残留导致的刷新异常。

## 1.0.6 - 2026-06-14

- 修复首页切换线路偶尔未生效的问题。
- 优化线路名称显示读取逻辑，减少 UI 与实际选中线路短暂不同步。

## 1.0.5 - 2026-06-14

- 修复暂停按钮内描边超出按钮范围的问题。
- 优化浅色主题开始按钮底部边角显示，让按钮边缘更干净。

## 1.0.4 - 2026-06-14

- 增加 7 套主题，并将冰川蓝改为默认主题。
- 修复主题迁移映射，避免旧版本升级后错位。
- 优化主题选择页显示与滚动适配。

## 1.0.3 - 2026-06-12

- 优化首页仪表盘布局，让模块比例更均衡。
- 保持现有模块紧凑展示，不再强行铺满整个屏幕。
- 调整实时网速卡片、流量卡片、开始按钮和地区延迟卡片的间距。

## 1.0.2 - 2026-06-12

- 修复部分内置主题无法切换的问题。
- 优化右上角按钮在不同主题下的文字可见性。
- 将偏重的“秋叶”配色替换为更柔和的“浅秋”主题。
- 线路管理支持单条线路编辑和删除。
- 优化线路名称和链接输入框的文字对齐。

## 1.0.1 - 2026-06-12

- 修正 Release 标签与 README 下载链接的对应关系。
- 应用功能无变更。

## 1.0.0 - 2026-06-12

- 首个开源版本。
- 原生 Android 实现。
- 实时显示 `MB/s` 和 `Mbps`。
- 支持总流量和本次流量统计。
- 支持流量线路选择和跑流量控制。
- 支持线程数、速率上限、流量上限、锁屏运行、增强并发和通知栏设置。
- 支持多主题切换。
