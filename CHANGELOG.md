# Changelog

## 1.0.14 - 2026-06-22

- 新增 24 个“时景”手动主题，可自由选择自动流转主题里的任意小时配色。
- 保留“时景流转”按真实时间自动切换，手动时景主题不再随时间变化。

## 1.0.13 - 2026-06-17

- 将“时景流转”动态主题扩展为 24 小时逐时配色，每小时变化更明显。
- 优化主题选择页说明，让动态主题的切换规则更清楚。

## 1.0.12 - 2026-06-17

- 新增“时景流转”动态主题，根据本地时间在黎明、日出、清晨、正午、午后、黄昏、暮光、夜晚、深夜和午夜配色间自动切换。
- 动态主题在前台每分钟检查时间段变化，跨时段后自动刷新界面。
## 1.0.11 - 2026-06-15

- 设置页线路管理新增一键导入节点，支持从剪贴板或手动粘贴批量导入。
- 导入节点支持纯链接、名称加链接、逗号或竖线分隔等常见格式，并自动跳过重复链接。

## 1.0.10 - 2026-06-14

- 软件处于前台时保持屏幕常亮，切到后台后恢复系统自动锁屏。
- Release 页面改为展示本次更新内容，不再重复功能列表。

## 1.0.9 - 2026-06-14

- 移除首页内置项目地址入口，让应用界面更专注。
- 保留 README 与 Release 页面中的开源项目链接。

## 1.0.8 - 2026-06-14

- 优化 GitHub 首页介绍，补充具体功能与下载安装说明。
- 清理公开文档中的非正式表述，让项目介绍更适合作为开源软件说明。

## 1.0.7 - 2026-06-14

- 修复重启后首页线路名称一直显示第一条的问题。
- 修复从设置页返回首页后线路视图引用残留导致的刷新异常。

## 1.0.6 - 2026-06-14

- 修复首页切换线路偶尔未生效的问题。
- 优化线路名称显示读取逻辑，减少 UI 与实际选中线路短暂不同步。

## 1.0.5 - 2026-06-14

- 修复暂停按钮内描边超出按钮范围的问题。
- 优化浅色主题开始按钮底部边角显示，让按钮边缘更干净。

## 1.0.4 - 2026-06-14

- 增加 7 套主题，并将冰川蓝改为默认主题。
- 修复主题迁移映射，避免旧版本升级后错位。
- 优化主题选择页显示与滚动适配。

## 1.0.3 - 2026-06-12

- 优化首页仪表盘布局，让模块比例更均衡。
- 保持现有模块紧凑展示，不再强行铺满整个屏幕。
- 调整实时网速卡片、流量卡片、开始按钮和地区延迟卡片的间距。

## 1.0.2 - 2026-06-12

- 修复部分内置主题无法切换的问题。
- 优化右上角按钮在不同主题下的文字可见性。
- 将偏重的“秋叶”配色替换为更柔和的“浅秋”主题。
- 线路管理支持单条线路编辑和删除。
- 优化线路名称和链接输入框的文字对齐。

## 1.0.1 - 2026-06-12

- 修正 Release 标签与 README 下载链接的对应关系。
- 应用功能无变更。

## 1.0.0 - 2026-06-12

- 首个开源版本。
- 原生 Android 实现。
- 实时显示 `MB/s` 和 `Mbps`。
- 支持总流量和本次流量统计。
- 支持测试线路选择和主控制按钮。
- 支持线程数、速率上限、流量上限、锁屏运行、增强并发和通知栏设置。
- 支持多主题切换。
