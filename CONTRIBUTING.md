# Contributing

Thanks for your interest in improving 网络面板.

## Development

1. Install Android Studio or Android SDK.
2. Clone the repository.
3. Build with:

```powershell
./gradlew assembleDebug
```

On Windows PowerShell:

```powershell
.\gradlew.bat assembleDebug
```

## Pull Requests

- Keep changes focused.
- Do not commit APK files, keystores, local screenshots, or personal configuration.
- Test with `assembleDebug` before submitting.
